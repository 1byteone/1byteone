ruoyi-ai Architecture

Enterprise AI application platform with multi-model access, Supervisor multi-agent orchestration, RAG, GraphRAG, and workflow composition.

Application Surface

  • - Vue 3 user portal and Vben admin console share one API entry
  • - ruoyi-admin exposes REST, SSE, and WebSocket interaction patterns
  • - Workflow builder supports interruption and human feedback loops

AI Core

  • - Supervisor coordinates Skills, WebSearch, SQL, and Chart agents
  • - ruoyi-chat anchors RAG, MCP tools, and multi-model routing
  • - langgraph4j drives visual AI workflow orchestration

Enterprise Data

  • - Milvus, Weaviate, or Qdrant provide vector persistence
  • - Neo4j enriches retrieval through GraphRAG traversal
  • - MySQL and Redis back business modules, tenancy, and cache flows
