ai-passage-creator 系统架构图

面向内容创作场景的多 Agent 流水线，围绕 StateGraph 编排、并行配图策略、SSE 流式回传、腾讯云 COS 与 Stripe 会员能力形成完整创作闭环。

人在回路

  • - 选题与大纲阶段保留作者审阅，确保内容方向可控
  • - SSE 持续回传中间状态，让编辑端实时感知生成进度
  • - 初稿与最终成品都可作为人工干预与确认节点

Agent 流水线

  • - 标题、大纲、正文、配图分析、并行配图与图文合并组成完整 StateGraph
  • - 配图策略会分流到 Gemini、图库、Mermaid、Iconify 等多条能力路径
  • - AOP 日志采集可追踪每个 Agent 的耗时、成功率与关键载荷

商业化闭环

  • - 腾讯云 COS 负责图片资产落盘与可分享 URL 回传
  • - Stripe Checkout 与 Webhook 校验承接终身 VIP 购买链路
  • - 会员能力在不破坏兜底策略的前提下开放更高阶配图方案
