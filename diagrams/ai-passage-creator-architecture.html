ai-passage-creator Architecture

Human-in-the-loop content production pipeline with StateGraph agents, parallel image strategies, SSE streaming, Tencent COS, and Stripe membership lifecycle.

Human-in-the-Loop

  • - Topic selection and outline review stay in the author feedback loop
  • - SSE streams intermediate agent state back to the editor UI
  • - Draft and final output are both visible checkpoints

Agent Pipeline

  • - Title, Outline, Content, Analyzer, ParallelImage, and Merger form one StateGraph
  • - Strategy routing fans out to Gemini, stock, diagram, icon, and fallback paths
  • - AOP logging captures agent latency, success, and payload traces

Commercial Loop

  • - Tencent COS stores generated image assets and returns shareable URLs
  • - Stripe Checkout and Webhook verification manage permanent VIP upgrades
  • - Premium membership gates advanced image strategies while keeping fallback paths
