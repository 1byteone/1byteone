mewpaw-code 系统架构图

基于 Java 21 的命令行编码智能体，通过 ReAct 循环、安全过滤链、工具注册表、MCP 扩展与多模态交互界面形成完整的执行运行时。

交互模式

  • - CLI、REPL、全屏 TUI 与 React Web 工作台共享统一事件模型
  • - Agent 事件可通过终端与 SSE 视图近实时流式回传
  • - 交互表面与工具执行内核解耦，便于扩展不同前端形态

Agent 运行时

  • - ReAct 循环围绕 Think、Act、Observe 执行，并受重试上限约束
  • - 工具注册表路由 6 个内置工具，并可挂接 MCP 外部工具
  • - 执行钩子补充校验、流式输出、计时与取消控制

安全模型

  • - ToolFilter、PathGuard、扫描、确认与审计组成单一责任链
  • - ContextCompressor 与会话持久化确保长会话仍可控、可追踪
  • - 危险操作可在真正执行前被阻断、暂停或要求人工确认
