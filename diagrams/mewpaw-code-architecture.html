mewpaw-code Architecture

Java 21 coding agent with ReAct loop, layered security chain, built-in tool registry, MCP extension path, and multimodal interaction surfaces.

Interaction Modes

  • - CLI, REPL, fullscreen TUI, and React web workspace share one event model
  • - Agent events stream through terminal and SSE views in near real time
  • - Interaction surfaces stay separate from the tool execution core

Agent Runtime

  • - ReAct loop cycles through Think, Act, and Observe with bounded retries
  • - ToolRegistry routes six built-in tools plus MCP-backed external tools
  • - Hooks add validation, streaming, timing, and cancellation controls

Safety Model

  • - ToolFilter, PathGuard, scanner, confirmation, and audit act as one chain
  • - ContextCompressor and transcript persistence keep long sessions manageable
  • - Dangerous operations can be denied or paused before tool execution begins
